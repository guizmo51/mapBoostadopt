A propos

Ce projet n'a aucun but commercial. Il s'agit juste d'une cartographie des utilisateurs d'AdopteUnmec.

Ce projet utilise simplement un robot qui enregistre les profils, toutes les données visibles içi ne sont issues que de ce crawling et d'un travail manuel sur ces données.

Ce projet se veut respectueux des lois en vigueur mais aussi du respect de la vie privée et des libertés. Si votre profil est présent içi et que vous souhaitez le faire disparaître merci de nous contacter

Ce projet utilise les technologies suivantes :

  • NodeJS
  • ExpressJS
  • MongoDB
  • Jquery
  • Mapquest
  • OpenStreetMap
  • Leaflet
  • Bootstrap
